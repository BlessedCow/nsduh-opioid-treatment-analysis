# Opioid Treatment Gap Analysis (NSDUH 2024)

## Overview
This project analyzes the 2024 National Survey on Drug Use and Health (NSDUH) to examine opioid use, severity, and treatment utilization in the United States.

## Key Question
How does opioid use disorder severity relate to treatment and medication exposure?

## Key Findings

- Only ~3% of past-year opioid users received treatment  
- Treatment rates increase with severity but remain low overall  
- Even among higher-severity users, many individuals did not receive treatment  
- Medication indicators (buprenorphine, methadone) increase with severity but are not universal  

## Interpretation
The results suggest a significant treatment gap in opioid use disorder, including among individuals with higher severity levels.

## Data Source
NSDUH 2024 Public Use File (SAMHSA)

## Tools Used
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Notes
This analysis is descriptive and based on self-reported survey data. Medication variables represent indicators and may not fully capture formal treatment.
